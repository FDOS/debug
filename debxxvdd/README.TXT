
 About DEBXXVDD
 
 DEBXXVDD is a "Virtual DOS Driver" (VDD) for Windows XP/Vista NTVDM.
 It provides additional features for DOS applications, mainly for DOS
 debuggers. Some of the features will be available only if the user 
 has administrator rights.
 
 Installation
 
 DEBXXVDD.DLL must be copied to a directory included in the PATH
 environment variable. AFAIK, for Vista, it has to be copied to the
 Windows system directory.
